# Security Policy

## Supported Versions

these are the supported versions.

|    Version    |      Supported     |
| ------------- | ------------------ |
| Current       | :white_check_mark: |
| Anything else | :x:                |

## Reporting a Vulnerability

go to the issues page and mark the end of your title with "(Security Vulnerability)" (no the parentheses aren't a placeholder for you to put something there.). expect a response in a day or two, maybe a week or a month if i'm busy, expect duct tape fixes at worst if the vulnerability is accepted.
