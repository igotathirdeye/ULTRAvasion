ULTRAvasion

ULTRAvasion is a free and open source FPS/RPG game developed in C built using CMake.

How ULTRAvasion was born

ULTRAvasion was actually supposed to be made in Unity, however my PC was so shit that it couldn't even open a Unity project.
I gave Unreal Engine a shot, and it crashed the second I booted it up.

(I don't remember much of this next part)

So then, I realized I should give C a shot.
And the rest is history.

Before ULTRAvasion was ULTRAvasion

While the oldest public version on GitHub says it was made in Jan 12 of 2026, the project actually started at about mid-November to early-December of 2025.
During that time, it was called "untitled evasion game".
Back then, the game was just a C program that returned 0.
Then, in late-December of 2025, I picked this project back up and quickly started working on it.
I then decided to change the name to ULTRAvasion (ULTRA from ULTRAKILL and vasion from "untitled evasion game")
